== Description ==

This plugin lets you import data from csv files to WordPress. This version does not
control custom fields match if they have not been used, and also you have to delete 
all post info before you use this plugin.
The CSV file must be saved in UTF-8. Otherwise, all string with special characters 
won't be saved.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `yk-csv-to-wp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
